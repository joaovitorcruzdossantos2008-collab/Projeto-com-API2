Consulta de Criptomoedas com Java + API CoinGecko
Projeto desenvolvido em Java utilizando HttpClient, HttpRequest e HttpResponse para consumir a API da CoinGecko.
📊 Funcionalidade
O usuário informa o nome de uma criptomoeda e o sistema:
Realiza requisição HTTP à API
Processa o JSON retornado
Exibe a cotação atual da criptomoeda
Mostra o valor em moeda escolhida (ex: BRL ou USD)
🚀 Tecnologias utilizadas
Java 
HttpClient
API 
Manipulação de JSON
🎯 Objetivo do projeto
Praticar:
Integração com APIs públicas
Consumo de dados em tempo real
Manipulação de JSON
Requisições HTTP em Java
